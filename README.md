# 🚀 SM Technology - Frontend Job Task

This project was developed as part of a **Frontend Developer Job Task** for **SM Technology**.  
The objective of this task was to convert the provided **Figma design** into a fully functional and responsive website using **HTML, CSS, and JavaScript**.

The project focuses on **clean UI implementation, responsive design, and structured frontend development practices**.

## ✨ Features

- Pixel-perfect UI based on the provided design  
- Fully responsive for **mobile, tablet, and desktop**  
- Clean and organized code structure  
- Responsive navigation menu with **hamburger toggle**  
- Optimized layout using **modern CSS techniques**  
- Smooth user experience across different devices
